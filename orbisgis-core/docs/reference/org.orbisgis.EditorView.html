org.orbisgis.EditorView

Description:

This extension point is used to add new panels inside the main frame of the application that are associated with an specific editor

Example:

    < extension point=" org.orbisgis.EditorView" >
        < editor-view id=" org.orbisgis.editorViews.Toc" class=" org.orbisgis.editorViews.toc.TocView" editor-id=" org.orbisgis.editors.Map" icon=" /org/orbisgis/images/map.png" title=" TOC" />
    </ extension>

editor-view

Defines a view to be included in the main frame of the application

Attributes:

Attribute nameRequiredDescription
class optional Class that implements the org.orbisgis.views.IEditorView
icon optional Icon to be shown in the view
title optional Title to display in the view
editor-id required Identifier of the associated editor. The view will only be available when an editor with this id is the active editor
id required Identifier for internal use. Should be unique
