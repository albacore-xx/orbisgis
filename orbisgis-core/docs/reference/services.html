Services in OrbisGIS

This is the description of the available services in OrbisGIS. An OrbisGIS service is just a java instance that is accessible by a name and that implements a specified interface. From a practical point of view, services will be the entry point to access the different functionalities in OrbisGIS

Service nameService interfaceDescription
interface org.orbisgis.ApplicationInfoorg.orbisgis.ApplicationInfoGets information about the application: name, version, etc.
interface org.orbisgis.DataManagerorg.orbisgis.DataManagerAccess to the sources, to its properties (indexes, etc.) and its contents, either raster or vectorial
interface org.orbisgis.configuration.BasicConfigurationorg.orbisgis.configuration.BasicConfigurationManages the basic configurations (key, value)
interface org.orbisgis.editors.map.MapContextManagerorg.orbisgis.editors.map.MapContextManagerGives access to the current MapContext
interface org.orbisgis.errorManager.ErrorManagerorg.orbisgis.errorManager.ErrorManagerNotification of errors to the system
interface org.orbisgis.geocognition.Geocognitionorg.orbisgis.geocognition.GeocognitionRegistry containing all the artifacts produced and shared by the users
interface org.orbisgis.javaManager.JavaManagerorg.orbisgis.javaManager.JavaManagerExecution of java code and java scripts
interface org.orbisgis.map.export.MapExportManagerorg.orbisgis.map.export.MapExportManagerManages the export of MapContexts to different formats.
interface org.orbisgis.outputManager.OutputManagerorg.orbisgis.outputManager.OutputManagerService to send messages to the output system
interface org.orbisgis.pluginManager.PluginManagerorg.orbisgis.pluginManager.PluginManagerGives access to the plug-in system manager. It can stop the application, access the log files, etc.
interface org.orbisgis.pluginManager.background.BackgroundManagerorg.orbisgis.pluginManager.background.BackgroundManagerExecute tasks in background processes, showing progress bars. Gives access to the job queue
interface org.orbisgis.renderer.legend.carto.LegendManagerorg.orbisgis.renderer.legend.carto.LegendManagerManages the list of available legend types
interface org.orbisgis.renderer.symbol.SymbolManagerorg.orbisgis.renderer.symbol.SymbolManagerManages the list of available symbol types
interface org.orbisgis.updates.UpdateManagerorg.orbisgis.updates.UpdateManagerService to install updates from a remote site
interface org.orbisgis.view.ViewManagerorg.orbisgis.view.ViewManagerOpen, close and gets access to the views
interface org.orbisgis.views.editor.EditorManagerorg.orbisgis.views.editor.EditorManagerGets access to the active editor and its document
interface org.orbisgis.windows.mainFrame.UIManagerorg.orbisgis.windows.mainFrame.UIManagerGets access to the main frame
interface org.orbisgis.workspace.OGWorkspaceorg.orbisgis.workspace.OGWorkspaceGives access to directories inside the workspace. You can use the temporal folder in the workspace through this service. It lets the access to the results folder
interface org.orbisgis.workspace.Workspaceorg.orbisgis.workspace.WorkspaceChange workspace, save files in the workspace, etc.
