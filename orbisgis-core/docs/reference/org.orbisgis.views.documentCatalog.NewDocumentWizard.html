org.orbisgis.views.documentCatalog.NewDocumentWizard

Description:

This extension point can be extended to add new types of documents to the document catalog

Example:

    < extension id=" org.orbisgis.views.documentCatalog.newDocumentWizards.NewMapDocument" point=" org.orbisgis.views.documentCatalog.NewDocumentWizard" >
        < wizard id=" org.orbisgis.views.documentCatalog.newDocumentWizards.NewMapDocument" class=" org.orbisgis.views.documentCatalog.newDocumentWizards.NewMapDocument" />
    </ extension>
