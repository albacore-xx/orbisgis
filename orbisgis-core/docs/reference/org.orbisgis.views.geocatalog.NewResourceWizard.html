org.orbisgis.views.geocatalog.NewResourceWizard

Description:

This extension point can be extended to add new types of sources to the catalog

Example:

    < extension point=" org.orbisgis.views.geocatalog.NewResourceWizard" >
        < wizard id=" org.orbisgis.views.geocatalog.newResourceWizards.NewFile" class=" org.orbisgis.views.geocatalog.newResourceWizards.file.NewFile" />
    </ extension>
