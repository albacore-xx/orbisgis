org.orbisgis.views.geocognition.Filter

Description:

This extension point is used to add filters to the geocognition. A button will be installed that will filter the contents when the user clicks on it

Example:

    < extension point=" org.orbisgis.views.geocognition.Filter" id=" org.orbisgis.views.geocognition.filter.Basic" >
        < filter id=" org.orbisgis.views.geocognition.filters.Map" icon=" /org/orbisgis/images/map.png" class=" org.orbisgis.views.geocognition.filters.Map" />
        < filter id=" org.orbisgis.views.geocognition.filters.SQL" icon=" /org/orbisgis/images/script_code.png" class=" org.orbisgis.views.geocognition.filters.SQL" />
        < filter id=" org.orbisgis.views.geocognition.filters.Symbology" icon=" /org/orbisgis/images/palette.png" class=" org.orbisgis.views.geocognition.filters.Symbology" />
    </ extension>
