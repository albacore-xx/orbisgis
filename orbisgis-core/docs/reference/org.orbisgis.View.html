org.orbisgis.View

Description:

This extension point is used to add new panels inside the main frame of the application

Example:

    < extension point=" org.orbisgis.View" >
        < view id=" org.orbisgis.views.Geocatalog" icon=" /org/orbisgis/images/geocatalog.png" class=" org.orbisgis.views.geocatalog.CatalogView" title=" Geocatalog" />
    </ extension>

Content:

view

Defines a view to be included in the main application frame

Attributes:

Attribute nameRequiredDescription
class optional Class that implements the org.orbisgis.view.IView that will contain the logic to add the new view
icon optional Icon to be shown in the view
title optional Title to display in the view
isEditor optional True if this view contains all the editors in the system. Only one view can have this attribute to true. If this attribute is true, the class must implement org.orbisgis.view.IEditorsView instead of org.orbisgis.view.IView
id required Identifier for internal use. Should be unique
