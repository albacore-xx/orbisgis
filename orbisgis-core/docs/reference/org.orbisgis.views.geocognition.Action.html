org.orbisgis.views.geocognition.Action

Description:

This extension point is used to add contextual actions in the geocognition. This actions will be shown when the user right clicks the geocognition

Example:

    < extension point=" org.orbisgis.views.geocognition.Action" id=" org.orbisgis.views.geocognition.actions.Basic" >
        < menu text=" Add..." id=" org.orbisgis.views.geocognition.New" />
        < action id=" org.orbisgis.views.geocognition.actions.Clear" text=" Clear" menu-group=" org.orbisgis.views.geocognition.Remove" class=" org.orbisgis.views.geocognition.actions.Clear" />
        < action id=" org.orbisgis.views.geocognition.actions.Open" text=" Open" icon=" /org/orbisgis/images/application_go.png" menu-group=" org.orbisgis.views.geocognition.Basic" class=" org.orbisgis.views.geocognition.actions.Open" />
        < action id=" org.orbisgis.views.geocognition.actions.Remove" text=" Remove" icon=" /org/orbisgis/images/remove.png" menu-group=" org.orbisgis.views.geocognition.Remove" class=" org.orbisgis.views.geocognition.actions.Remove" />
        < action id=" org.orbisgis.views.geocognition.actions.NewElement" text=" Element..." icon=" /org/orbisgis/images/application_go.png" menu-group=" org.orbisgis.views.geocognition.Basic" class=" org.orbisgis.views.geocognition.actions.NewElement" menu-id=" org.orbisgis.views.geocognition.New" />
    </ extension>

menu

Installs a parent menu

Attributes:

Attribute nameRequiredDescription
id required Id of the menu, referenced by other attributes
text required Text to be shown in the menu
parent optional References the id of another menu
icon optional Path to the icon to show in the menu. It has to be in the classpath of the plug-in

action

Defines a contextual action for the geocognition.

Attributes:

Attribute nameRequiredDescription
id required id of the action. For internal use. Must be unique
menu-group required The group of the action. It will be used to group graphically the actions that have the same value in this attribute
menu-id optional Id of the menu where this action will be installed
text required Text to display to the user
icon optional Icon to display to the user
class required Class that implements the action. It has to implement org.orbisgis.views.geocognition.action.IGeocognitionAction or org.orbisgis.views.geocognition.action.IGeocognitionGroupAction
