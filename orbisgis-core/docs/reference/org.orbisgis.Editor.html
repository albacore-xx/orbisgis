org.orbisgis.Editor

Description:

This extension point is used to add new editors inside geoview

Example:

    < extension id=" org.orbisgis.editor.Map" point=" org.orbisgis.Editor" >
        < editor id=" org.orbisgis.editors.Map" icon=" /org/orbisgis/images/map.png" class=" org.orbisgis.editors.map.MapEditor" />
    </ extension>

editor

Adds an editor to the system
