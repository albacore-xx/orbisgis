org.orbisgis.editor.Action

Description:

Action associated to an editor that will be installed in the menu and toolbar of the application

Example:

    < extension id=" org.orbisgis.editors.map.actions.NewLayer" point=" org.orbisgis.editor.Action" >
        < menu id=" org.orbisgis.Map" text=" Map" icon=" /org/orbisgis/images/map.png" />
        < editor-action menu-id=" org.orbisgis.Map" id=" org.orbisgis.editors.map.actions.NewLayer" text=" New Layer..." icon=" /org/orbisgis/images/add.png" menu-group=" org.orbisgis.editors.map.actions.LayerActionGroup" editor-id=" org.orbisgis.editors.Map" class=" org.orbisgis.editors.map.actions.NewLayer" />
    </ extension>

Content:

Element nameminOccursmaxOccurs
menu0unbounded
toolbar0unbounded
editor-action1unbounded

Attributes:

Attribute nameRequiredDescription
point required Id of the extension point
id optional Id of the extension. For debuging purposes only
name optional Name of the extension. For debuging purposes only

toolbar

Installs a toolbar. A tool bar from other actions can be used.

Attributes:

Attribute nameRequiredDescription
id required Id of the toolbar, referenced by the action attributes
text required Text to be shown in the tool bar

menu

Installs a parent menu

Attributes:

Attribute nameRequiredDescription
id required Id of the menu, referenced by other menus and by the editor-action element
text required Text to be shown in the menu
parent optional References the id attribute of another menu
icon optional Path to the icon to show in the menu. It has to be in the classpath of the plug-in

editor-action

Installs an action in the main frame of the application that is associated with a concrete editor. This action will only be available when the current editor is the one specified by the editor-id attribute

Attributes:

Attribute nameRequiredDescription
class required Class that implements the org.orbisgis.view.editor.IEditorAction interface
id required Id of the action. Must be unique
text required Text that will appear in the menu that triggers this action
menu-id optional Id of the menu where this action will be installed
icon optional Icon associated to the action
menu-group optional The group to associate the actions in the menu. Different groups will be separated visually
toolbar-id optional Id of the toolbar where this action will be installed. It is also necessary to specify the icon attribute
selectable optional If the action is selectable. If this attribute exists and it's set to true the interface to implement by the class specified in the 'class' attribute changes to org.orbisgis.views.editor.IEditorSelectableAction
editor-id required The id of the associated editor. This action will be actived only if the current editor has an id equals to this attribute.
