org.orbisgis.editorViews.toc.Action

Description:

This extension point is used to add contextual actions in the layer tree. This actions will be shown when the user right clicks in a layer

Example:

    < extension point=" org.orbisgis.editorViews.toc.Action" >
        < action id=" org.edit" text=" Edit Legend" menu-group=" editlayer" icon=" /org/orbisgis/images/palette.png" class=" org.orbisgis.editorViews.toc.actions.cui.EditLayerAction" />
    </ extension>

Attribute nameRequiredDescription

icon optional Path to the icon to show in the menu. It has to be in the classpath of the plug-in

action

Defines a contextual action of the toc in the geoview.

Attributes:

Attribute nameRequiredDescription
id required id of the action. For internal use. Must be unique
menu-group required The group of the action. It will be used to group graphically the actions that have the same value in this attribute
menu-id optional Id of the menu where this action will be installed
text required Text to display to the user
icon optional Icon to display to the user
class required Class that implements the action. It has to implement org.orbisgis.editorViews.toc.action.ILayerAction. If the action needs to deal with more than one layer at the same time it can implement org.orbisgis.editorViews.toc.action.IMultipleLayerAction instead
