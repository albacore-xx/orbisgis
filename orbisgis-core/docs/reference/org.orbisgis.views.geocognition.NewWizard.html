org.orbisgis.views.geocognition.NewWizard

Description:

This extension point can be extended to add new elements to the geocognition

Example:

    < extension id=" org.orbisgis.views.geocognition.wizards.NewRegisteredSQLArtifacts" point=" org.orbisgis.views.geocognition.NewWizard" >
        < category id=" org.orbisgis.views.geocognition.Processing" name=" Processing" />
        < wizard id=" org.orbisgis.views.geocognition.wizards.NewRegisteredSQLArtifacts" category=" org.orbisgis.views.geocognition.Processing" class=" org.orbisgis.views.geocognition.wizards.NewRegisteredSQLArtifact" />
    </ extension>
