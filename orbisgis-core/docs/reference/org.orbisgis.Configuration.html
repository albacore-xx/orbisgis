org.orbisgis.Configuration

Description:

Configuration that will be installed in the preferences dialog

Example:

    < extension point=" org.orbisgis.Configuration" >
        < group id=" org.orbisgis.Updates" text=" Updates" />
        < configuration id=" org.orbisgis.configurations.ProxyConfiguration" text=" Proxy" class=" org.orbisgis.configurations.ProxyConfiguration" group-id=" org.orbisgis.Updates" />
        < configuration id=" org.orbisgis.configurations.UpdatesConfiguration" text=" Updates" class=" org.orbisgis.configurations.UpdatesConfiguration" group-id=" org.orbisgis.Updates" />
    </ extension>

Content:

Element nameminOccursmaxOccurs
group1unbounded
configuration0unbounded

Attributes:

Attribute nameRequiredDescription
point required Id of the extension point
id optional Id of the extension. For debugging purposes only
name optional Name of the extension. For debugging purposes only

group

Installs a group in the configuration tree

Attributes:

Attribute nameRequiredDescription
id required Id of the group, referenced by other attributes
text required Text to be shown in the group
parent optional References the id of another group

configuration

Installs a configuration tab

Attributes:

Attribute nameRequiredDescription
id required Id of the configuration
text required Text to be shown in the dialog
group-id required Parent of the configuration element
class required Class that implements the org.orbisgis.config.IConfig interface
