org.orbisgis.Action

Description:

Action that will be installed in the menu, tool bar, etc. of the OrbisGIS frame

Example:

    < extension id=" org.orbisgis.BasicActions" point=" org.orbisgis.Action" >
        < menu id=" org.orbisgis.File" text=" File" icon=" /org/orbisgis/images/house.png" />
        < menu id=" org.orbisgis.Geocatalog" text=" Geocatalog" icon=" /org/orbisgis/images/geocatalog.png" />
        < menu id=" org.orbisgis.Help" text=" Help" icon=" /org/orbisgis/images/help.png" />
        < toolbar id=" org.orbisgis.MainBar" text=" OrbisGIS" />
        < action id=" org.orbisgis.actions.Exit" text=" Exit" menu-id=" org.orbisgis.File" icon=" /org/orbisgis/images/exit.png" toolbar-id=" org.orbisgis.MainBar" class=" org.orbisgis.actions.Exit" />
        < action id=" org.orbisgis.actions.About" text=" About OrbisGIS" menu-id=" org.orbisgis.Help" class=" org.orbisgis.actions.about.About" />
        < action id=" org.orbisgis.actions.Update" text=" Updates..." menu-id=" org.orbisgis.Help" class=" org.orbisgis.actions.Update" />
        < action id=" org.orbisgis.actions.NewResource" text=" New Resource" menu-id=" org.orbisgis.Geocatalog" icon=" /org/orbisgis/images/add.png" class=" org.orbisgis.actions.NewResource" />
    </ extension>

Content:

Element nameminOccursmaxOccurs
menu0unbounded
toolbar0unbounded
action1unbounded

Attributes:

Attribute nameRequiredDescription
point required Id of the extension point
id optional Id of the extension. For debuging purposes only
name optional Name of the extension. For debuging purposes only

toolbar

Installs a toolbar

Attributes:

Attribute nameRequiredDescription
id required Id of the toolbar, referenced by other attributes
text required Text to be shown in the tool bar

menu

Installs a parent menu

Attributes:

Attribute nameRequiredDescription
id required Id of the menu, referenced by other attributes
text required Text to be shown in the menu
parent optional References the id of another menu
icon optional Path to the icon to show in the menu. It has to be in the classpath

action

Installs an action in the OrbisGIS frame

Attributes:

Attribute nameRequiredDescription
class required Class that implements the org.orbisgis.action.IAction interface
id required Id of the action. Must be unique
text required Text that will appear in the menu that triggers this action
menu-id optional Id of the menu where this action will be installed
icon optional Icon associated to the action
menu-group optional The group to associate the actions in the menu. Different groups will be separated visually
toolbar-id optional Id of the toolbar where this action will be installed. It is also necessary to specify the icon attribute
selectable optional If the action is selectable. If this attribute is set to true the interface to implement by the class attribute becomes org.orbisgis.action.ISelectableAction
