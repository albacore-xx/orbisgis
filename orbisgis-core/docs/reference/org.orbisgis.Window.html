org.orbisgis.Window

Description:

Installs top level windows

Example:

    < extension point=" org.orbisgis.Window" >
        < window id=" org.orbisgis.MainFrame" newOnStartup=" true" class=" org.orbisgis.windows.mainFrame.OrbisGISFrame" />
    </ extension>

Content:

Element nameminOccursmaxOccurs
window1unbounded

Attributes:

Attribute nameRequiredDescription
point required Id of the extension point
id optional Id of the extension. For debuging purposes only
name optional Name of the extension. For debuging purposes only

window

Adds a top level window

Attributes:

Attribute nameRequiredDescription
id required Id of the window
newOnStartup optional If a new empty window is created when the system starts for the first time
class required Class that implements the IWindow interface
